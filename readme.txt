开发环境：
该系统在Windows11系统通过devc++5.11 及Visual Studio 1.78.2编写。
程序开发语言主要为c++，其中c++标准应在c++11（支持auto，成员函数const等特性）及以后。

链接要求：
由于本系统的项目文件为devc++产生的.dev文件，因此建议使用devc++打开本项目文件从而进入项目代码。
如果项目组织异常，可以手动将所有.h文件，.cpp文件添加至一个项目中再运行本项目。

运行要求：
由于代码通过相对路径与数据文件相联系，请保持本项目的Bank_Management_Syste_Code文件夹与Data文件夹的相对关系，
否则会导致文件读写异常，无法打开用户表文件，管理员表文件，以及交易记录文件。
此外，本系统的代码文件，数据文件均采用GBK编码模式（具体应是GB2312），其中数据文件用记事本打开后右下角应为ANSI标识。
在修改数据文件时请不要修改其编码方式，且不要修改deals文件夹内数据，否则可能导致程序输出错误或乱码等异常。

使用要求：
由于本系统需要使用账号密码登录才能进入，因此用户在使用前可简单查看Data文件夹内Users.txt，Admins.txt查询银行当前所含用户，管理员信息，并使用用户id及其密码登录。（由于各用户数据信息差异过大可能出现格式不齐，敬请见谅）
